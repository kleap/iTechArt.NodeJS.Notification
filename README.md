### Install: ###
2. type "npm install". It will install all npm modules from package.json settings.

### Commands: ###
* "npm run build" - building to /dist folder
* "npm start" - building and starting server in browser at "localhost:8080"
* "npm run server-only" - running server only on "localhost:8080"
* "npm run dev" - running webpack-dev-server only on "localhost:5000"
